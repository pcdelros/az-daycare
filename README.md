# Sonoran Sprouts Early Learning Website

A clean, responsive static website concept for a daycare located in Phoenix, Arizona.

## What is included

- `index.html` — homepage with hero section, trust highlights, program cards, and call-to-action sections
- `about.html` — care philosophy and center values
- `programs.html` — infant, toddler, preschool, and pre-k overview sections
- `enrollment.html` — tour and enrollment process page
- `contact.html` — tour request form layout and contact details
- `assets/css/styles.css` — responsive warm desert-inspired design system
- `assets/js/main.js` — mobile navigation, current year, and demo form feedback
- `robots.txt` and `sitemap.xml` — basic SEO support for GitHub Pages

## Design direction

The site uses a warm Phoenix/desert-inspired style instead of generic gradients or fake dashboard elements. It is designed to feel calm, trustworthy, parent-friendly, and easy to scan on mobile.

## Important placeholders to replace

Before publishing this as a real daycare website, update:

- Daycare name
- Phone number
- Email address
- Street address
- Hours of operation
- Tuition and fees
- Licensing details
- Staff/team information
- Real photos
- Real form handling service
- Final website domain in `sitemap.xml`

## Local preview

Open `index.html` in a browser. No build step is required.

## GitHub Pages

This is a static website and should work with GitHub Pages. In the repository settings, enable GitHub Pages from the `main` branch and root folder.

## Notes

The contact form currently shows a demo confirmation message only. Connect it to a real backend or form service before using it with customers.
